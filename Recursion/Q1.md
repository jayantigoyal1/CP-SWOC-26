# 932B – Recursive Queries

**Difficulty:** Easy–Medium  
**Tags:** Recursion, Precomputation, DP  

## Problem Summary
Define a recursive function `f(x)`:
- If `x < 10`, then `f(x) = x`
- Otherwise, `f(x) = f(product of non-zero digits of x)`

For multiple queries, each query gives `l, r, k`.  
You need to count how many numbers `x` in range `[l, r]` satisfy `f(x) = k`.

## Input
- First line: integer `q` (number of queries)
- Next `q` lines: `l r k`

## Output
For each query, print the count.

## Hint
- Precompute `f(x)` for all `x` up to `10^6`
- Use prefix sums for fast range queries
